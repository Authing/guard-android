## Authing 控制台配置

在控制台的创建自建应用时，请选择客户端应用，其它配置注意：

- **授权配置**：`授权模式` 选中 `authorization_code`、 `implicit`、 `password`、`refresh_token`、`authing_token`
- **授权配置**：`返回类型` 选中 `code`
